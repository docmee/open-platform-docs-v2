## 积分消耗规则

### 模板型 PPT

- 每生成 1 份模板型 PPT，消耗 **1 积分**。

### HTML 型 PPT

- HTML 型（新）PPT 按页计费，每页消耗 **0.8 积分**。
- 最终结算时，按整份 PPT 的总价**向下取整**扣费。

例如，生成 1 份 16 页的 HTML 型 PPT：

```text
16 × 0.8 = 12.8
向下取整后，实际扣费 12 积分
```

### 文多多智能体接入

- 如果您使用文多多智能体进行文档大纲推理，每一整轮循环调用将消耗 **2 积分**。
- 一整轮循环调用包括：**思考 + 搜索 + 阅读 + 整理 + 大纲写作**。

### 自定义模板

- 传统自定义模板：每次消耗 **2 积分**。
- HTML 型（新）自定义模板：按模板结构消耗 **5 - 9 积分**。

具体规则如下：

- 仅包含内容页的 HTML 型自定义模板：消耗 **5 积分**。
- 若包含功能页，则在 5 积分基础上额外累加。
- 功能页包括：**首页、尾页、章节页、目录页**。
- 每增加 1 页功能页，额外增加 **1 积分**。

例如：

- 仅内容页模板：5 积分
- 内容页 + 首页：6 积分
- 内容页 + 首页 + 尾页：7 积分
- 内容页 + 首页 + 尾页 + 章节页 + 目录页：9 积分
