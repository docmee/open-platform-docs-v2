# 能力介绍

`HTML 转 PPTX` 能力用于接收一组按顺序排列的 HTML 页面字符串，并将它们组合转换为一个可下载的 PPTX 文件。

## 适用场景

- 将业务系统中已经渲染完成的报告页面导出为 PPT。
- 将 H5、看板或可视化页面按页归档为演示文稿。
- 将服务端批量生成的 HTML 页面组合为标准 PPTX 交付文件。

## 输入输出概览

- 输入：<InlineCode>htmls: string[]</InlineCode>
- 输出：包含 <InlineCode>fileUrl</InlineCode> 的结果响应
- 结果文件：PPTX
- 返回方式：JSON 响应中返回 OSS 临时下载地址，不直接返回文件流

## 能力边界

- 服务端按 <InlineCode>htmls</InlineCode> 数组顺序生成 PPT 页顺序。
- 单页失败不一定导致整单失败，只要至少成功生成 1 页并产出文件，顶层仍返回 <InlineCode>code = 0</InlineCode>。
- 如果你的业务要求页级内容完全可控，建议同时阅读接入须知、完整 API 文档和最佳实践。
