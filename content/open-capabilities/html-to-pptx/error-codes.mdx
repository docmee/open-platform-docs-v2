# 错误码说明

本页用于集中说明 `HTML 转 PPTX` 能力的请求级错误、页级失败分类和顶层程序错误。

## 错误码总览

| 错误码 | 层级 | 说明 |
| --- | --- | --- |
| `40010` | 请求级 | 请求体结构或字段值不合法。 |
| `50101` | 页级 | 页面内容不满足转换要求，例如缺少 `.slide` 根元素。 |
| `50102` | 页级 | 页面约束或渲染失败，例如页面尺寸不一致。 |
| `50103` | 顶层程序错误 | 所有页面都失败，最终未生成可下载文件。 |
| `50199` | 顶层程序错误 | 服务内部程序异常。 |

## 40010

- 含义：请求体不合法
- 常见触发条件：
  - 请求体不是合法 JSON
  - <InlineCode>htmls</InlineCode> 缺失、不是数组或数组为空
  - <InlineCode>htmls</InlineCode> 中存在非字符串元素
  - <InlineCode>pageErrorMode</InlineCode> 不是 <InlineCode>blank</InlineCode> 或 <InlineCode>skip</InlineCode>
- 建议处理：修正请求结构后重新提交，不建议直接重试同一请求

## 50101

- 含义：页级内容不满足转换要求
- 常见触发条件：
  - HTML 中未找到 <InlineCode>.slide</InlineCode> 根元素
- 建议处理：修正对应页面 HTML 后重新提交，或在业务侧记录失败页并提示人工检查

## 50102

- 含义：页级约束或渲染失败
- 常见触发条件：
  - 某页尺寸与其他页面不一致
  - 单页渲染阶段异常，但不影响其他页面继续处理
- 建议处理：优先检查页面尺寸、样式和结构一致性

## 50103

- 含义：所有页面都失败，服务未生成最终文件
- 常见触发条件：
  - 本次请求中的所有页面都触发了页级失败
- 建议处理：检查 <InlineCode>errorDetails</InlineCode>，逐页修复后重新发起请求

## 50199

- 含义：服务内部程序异常
- 常见触发条件：
  - 服务端内部处理异常，无法完成转换
- 建议处理：记录 <InlineCode>requestId</InlineCode>，稍后重试；如持续出现，请联系平台支持
