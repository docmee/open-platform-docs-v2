# API 参考

这里汇总了 Docmee Open Platform 当前可用的全部 API，并按接入场景进行了重组。

## 阅读建议

- 新接入团队先看 [快速开始](/getting-started)
- 要快速浏览接口目录，先看 [全部 API 接口](/api-reference/all-api)
- 要联调具体参数，直接进入对应接口详情页

## 目录说明

- `鉴权与接入`：创建调用 Token 和通用鉴权前置能力
- `V2 生成流程`：官方推荐的主链路
- `V1 兼容接口`：历史方案、兼容接入和特殊生成模式
- `模板接口`：模板筛选与获取
- `PPT 获取与操作`：生成后的查询、下载、保存和编辑
- `自定义模板`：上传和管理私有模板
- `积分与记录`：计费统计、调用审计和全量查询
