# 分页查询模板

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/templates">https://open.docmee.cn/api/ppt/templates</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

分页查询系统模板或用户模板列表。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/templates \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "page": 1,
  "size": 10,
  "filters": {
    "type": 1,
    "category": null,
    "style": null,
    "themeColor": null
  }
}'
```

## 请求参数

```json
{
  "page": 1,
  "size": 10,
  "filters": {
    "type": 1,
    "category": null,
    "style": null,
    "themeColor": null
  }
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `page` | `number` | 否 | `1` | 分页查询参数。 |
| `size` | `number` | 否 | `10` | 分页查询参数。 |
| `filters` | `object` | 否 | `{"type":1,"category":null,"style":null,"themeColor":null}` | 模板筛选条件对象。 |
| `filters.type` | `number` | 否 | `1` | 模板筛选子字段。 |
| `filters.category` | `string` | 否 | `null` | 模板筛选子字段。 |
| `filters.style` | `string` | 否 | `null` | 模板筛选子字段。 |
| `filters.themeColor` | `string` | 否 | `null` | 模板筛选子字段。 |

## 关键字段

- `filters.type`：必传，`1` 为系统模板，`4` 为用户模板
- `coverUrl`：模板封面地址，访问时需要拼接 `?token=xxx`

## 使用说明

- 支持在 URL 上通过 `lang` 指定国际化语言
- 可与 [`template/options`](/api-reference/templates/template-options) 配合使用

import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class ListTemplatesExample {
  public static void main(String[] args) throws Exception {
    String body = """
      {
        "page": 1,
        "size": 10,
        "filters": {
          "type": 1,
          "category": null,
          "style": null,
          "themeColor": null
        }
      }
      """;

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://open.docmee.cn/api/ppt/templates"))
      .header("Content-Type", "application/json")
      .header("token", "YOUR_TEMP_TOKEN")
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}
