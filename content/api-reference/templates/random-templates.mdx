# 随机模板

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/randomTemplates">https://open.docmee.cn/api/ppt/randomTemplates</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

随机返回一批模板，适合做模板推荐、瀑布流加载或快速换一批。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/randomTemplates \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "size": 30,
  "filters": {
    "type": 1,
    "category": null,
    "style": null,
    "themeColor": null,
    "neq_id": []
  }
}'
```

## 请求参数

```json
{
  "size": 30,
  "filters": {
    "type": 1,
    "category": null,
    "style": null,
    "themeColor": null,
    "neq_id": []
  }
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `size` | `number` | 否 | `30` | 分页查询参数。 |
| `filters` | `object` | 否 | `{"type":1,"category":null,"style":null,"themeColor":null,"neq_id":[]}` | 模板筛选条件对象。 |
| `filters.type` | `number` | 否 | `1` | 模板筛选子字段。 |
| `filters.category` | `string` | 否 | `null` | 模板筛选子字段。 |
| `filters.style` | `string` | 否 | `null` | 模板筛选子字段。 |
| `filters.themeColor` | `string` | 否 | `null` | 模板筛选子字段。 |
| `filters.neq_id` | `array` | 否 | `[]` | 模板筛选子字段。 |

## 关键字段

- `size`：返回模板数量
- `neq_id`：排除已经出现过的模板 ID

## 使用说明

- 模板封面访问同样需要拼接 `?token=xxx`
- 支持通过 URL `lang` 参数切换多语言模板内容


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "VALUE"
  }
}
```

| 字段名 | 类型 | 示例 | 说明 |

| `data` | `object` | `{...}` | 业务数据对象。 |
