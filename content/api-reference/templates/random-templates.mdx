# 随机模板

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/randomTemplates`
- 鉴权：请求头传 `token`

## 接口作用

随机返回一批模板，适合做模板推荐、瀑布流加载或快速换一批。

## 请求参数

```json
{
  "size": 30,
  "filters": {
    "type": 1,
    "category": null,
    "style": null,
    "themeColor": null,
    "neq_id": []
  }
}
```

## 关键字段

- `size`：返回模板数量
- `neq_id`：排除已经出现过的模板 ID

## 使用说明

- 模板封面访问同样需要拼接 `?token=xxx`
- 支持通过 URL `lang` 参数切换多语言模板内容
