# 查询记录详情

## 接口信息

- 方法：<HttpMethodBadge method="GET" />
- 路径：<InlineCode value="https://open.docmee.cn/api/record/getById?id=">https://open.docmee.cn/api/record/getById?id=</InlineCode>
- 鉴权：请求头传 `Api-Key`

## 接口作用

获取单条积分记录对应的详细对象信息。

## 请求示例

```bash
curl --request GET \
  --url "https://open.docmee.cn/api/record/getById?id=" \
  --header "Api-Key: YOUR_API_KEY"
```

## 请求参数

- `id`：记录 ID

## 返回内容

可能包含：

- `name`
- `subject`
- `coverUrl`
- `fileUrl`
- `templateId`
- `createTime`

资源地址访问时通常仍需要拼接 `token`。
