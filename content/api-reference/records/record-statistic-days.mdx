# 按天统计积分使用

## 接口信息

- 方法：`POST`
- 路径：`/api/record/statisticDays`
- 鉴权：请求头传 `Api-Key`

## 接口作用

按天统计指定时间区间内的调用次数与积分消耗。

## 请求参数

```json
{
  "type": null,
  "uid": null,
  "startDate": "2025-01-01 00:00:00",
  "endDate": "2025-01-30 23:59:59"
}
```

## 返回字段

- `date`：日期
- `count`：使用次数
- `amount`：消耗积分
