# 查询积分使用记录

## 接口信息

- 方法：`POST`
- 路径：`/api/record/listPage`
- 鉴权：请求头传 `Api-Key`

## 接口作用

分页查询指定时间范围内的积分使用记录。

## 请求参数

```json
{
  "page": 1,
  "size": 100,
  "type": 1,
  "uid": "user_xxx",
  "startDate": "2025-01-01 00:00:00",
  "endDate": "2025-01-30 23:59:59"
}
```

## 关键字段

- `type=1`：PPT 生成
- `type=2`：模板上传

## 响应内容

每条记录通常包含 `id`、`type`、`amount`、`uid`、`createTime`。
