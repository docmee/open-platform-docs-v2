# 移除 Logo

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/removePptLogo">https://open.docmee.cn/api/ppt/removePptLogo</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

从 PPT 的指定页码或页面类型中移除 Logo。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/removePptLogo \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "pptId": "ppt_xxx",
  "pageIndex": 1,
  "pageTypes": [1, 2]
}'
```

## 请求参数

```json
{
  "pptId": "ppt_xxx",
  "pageIndex": 1,
  "pageTypes": [1, 2]
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `pptId` | `string` | 否 | `ppt_xxx` | 字段用途请结合接口场景传入。 |
| `pageIndex` | `number` | 否 | `1` | 字段用途请结合接口场景传入。 |
| `pageTypes` | `array` | 否 | `[1,2]` | 字段用途请结合接口场景传入。 |

## 页面类型

- `0`：首页
- `1`：目录页
- `2`：章节页
- `3`：内容页
- `4`：尾页
