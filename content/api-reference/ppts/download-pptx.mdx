# 下载 PPT

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/downloadPptx">https://open.docmee.cn/api/ppt/downloadPptx</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

返回 PPT 文件下载地址。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/downloadPptx \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "id": "ppt_xxx",
  "refresh": false
}'
```

## 请求参数

```json
{
  "id": "ppt_xxx",
  "refresh": false
}
```

- `id`：PPT ID
- `refresh`：是否重新刷新下载文件


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `string` | 否 | `ppt_xxx` | 业务对象 ID。 |
| `refresh` | `boolean` | 否 | `false` | 字段用途请结合接口场景传入。 |

## 响应示例

```json
{
  "code": 0,
  "data": {
    "id": "ppt_xxx",
    "name": "演示文稿名称",
    "subject": "演示主题",
    "fileUrl": "https://xxx"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `data` | `object` | `{"id":"ppt_xxx","name":"演示文稿名称","subject":"演示主题","fileUrl":"` | 业务数据对象。 |
| `data.id` | `string` | `ppt_xxx` | 对象唯一标识。 |
| `data.name` | `string` | `演示文稿名称` | 字段用途请结合接口场景传入。 |
| `data.subject` | `string` | `演示主题` | 字段用途请结合接口场景传入。 |
| `data.fileUrl` | `string` | `https://xxx` | 文件下载地址。 |

## 说明

`fileUrl` 默认有效期 2 小时。

import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class DownloadPptxExample {
  public static void main(String[] args) throws Exception {
    String body = "{\"id\":\"PPT_ID\",\"refresh\":false}";

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://open.docmee.cn/api/ppt/downloadPptx"))
      .header("Content-Type", "application/json")
      .header("token", "YOUR_TEMP_TOKEN")
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}
