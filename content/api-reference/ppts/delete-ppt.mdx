# 删除 PPT

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/delete">https://open.docmee.cn/api/ppt/delete</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

删除指定 PPT。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/delete \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "id": "ppt_xxx"
}'
```

## 请求参数

```json
{
  "id": "ppt_xxx"
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `string` | 否 | `ppt_xxx` | 业务对象 ID。 |

## 响应

成功时返回通用成功结构。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `data` | `object` | `{...}` | 业务数据对象。 |
