# 设置 Logo

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/addPptLogo">https://open.docmee.cn/api/ppt/addPptLogo</InlineCode>
- 鉴权：请求头传 `token`
- 请求体：`multipart/form-data`

## 接口作用

为指定 PPT 批量添加 Logo。


## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/addPptLogo \
  --header "token: YOUR_TEMP_TOKEN" \
  --form "pptId=PPT_ID" \
  --form "image=@demo.png" \
  --form "position=VALUE" \
  --form "pageIndex=VALUE"
```
## 主要参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `pptId` | `string` | 是 | PPT ID |
| `image` | `File` | 是 | `png` 或 `jpg` |
| `position` | `string` | 是 | 如 `TOP_RIGHT`、`BOTTOM_LEFT` |
| `pageIndex` | `integer` | 否 | 指定页码，从 1 开始 |
| `pageTypes` | `number[]` | 否 | 指定页面类型 |
| `marginX` | `number` | 否 | X 边距 |
| `marginY` | `number` | 否 | Y 边距 |
| `scale` | `number` | 是 | 缩放比例 |
| `rmOrgLogo` | `boolean` | 是 | 是否移除原有 Logo |

## 响应说明

成功后返回新的 `fileUrl` 下载地址。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "fileUrl": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `fileUrl` | `string` | `VALUE` | 文件下载地址。 |
