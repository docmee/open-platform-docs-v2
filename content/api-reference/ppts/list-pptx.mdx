# 获取 PPT 列表

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/listPptx">https://open.docmee.cn/api/ppt/listPptx</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

分页查询当前用户下的 PPT 作品列表。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/listPptx \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "page": 1,
  "size": 10
}'
```

## 请求参数

```json
{
  "page": 1,
  "size": 10
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `page` | `number` | 否 | `1` | 分页查询参数。 |
| `size` | `number` | 否 | `10` | 分页查询参数。 |

## 返回内容

每条记录通常包含 `id`、`subject`、`coverUrl`、`templateId`、`createTime` 等字段。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "id_xxx",
    "subject": "VALUE",
    "coverUrl": "VALUE",
    "templateId": "templateId_xxx",
    "createTime": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `id` | `string` | `id_xxx` | 对象唯一标识。 |
| `subject` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
| `coverUrl` | `string` | `VALUE` | 封面图地址。 |
| `templateId` | `string` | `templateId_xxx` | 模板 ID。 |
| `createTime` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
