# 加载 PPT 大纲内容

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/loadPptxMarkdown">https://open.docmee.cn/api/ppt/loadPptxMarkdown</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

获取某个 PPT 对应的 Markdown 大纲与内容。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/loadPptxMarkdown \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "id": "ppt_xxx",
  "format": "tree"
}'
```

## 请求参数

```json
{
  "id": "ppt_xxx",
  "format": "tree"
}
```

- `format=text`：返回 `markdownText`
- `format=tree`：返回 `markdownTree`


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `string` | 否 | `ppt_xxx` | 业务对象 ID。 |
| `format` | `string` | 否 | `tree` | 字段用途请结合接口场景传入。 |

## 适用场景

适合做编辑回显、结构分析或二次生成前的内容恢复。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `data` | `object` | `{...}` | 业务数据对象。 |
