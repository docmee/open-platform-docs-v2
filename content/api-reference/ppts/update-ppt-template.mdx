# 更换 PPT 模板

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/updatePptTemplate">https://open.docmee.cn/api/ppt/updatePptTemplate</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

为已经生成的 PPT 切换模板，并返回更新后的结构数据。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/updatePptTemplate \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "pptId": "ppt_xxx",
  "templateId": "template_xxx",
  "sync": false
}'
```

## 请求参数

```json
{
  "pptId": "ppt_xxx",
  "templateId": "template_xxx",
  "sync": false
}
```

- `sync=false`：默认异步更新，速度更快


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `pptId` | `string` | 否 | `ppt_xxx` | 字段用途请结合接口场景传入。 |
| `templateId` | `string` | 否 | `template_xxx` | 模板 ID。 |
| `sync` | `boolean` | 否 | `false` | 字段用途请结合接口场景传入。 |

## 响应说明

响应中会返回新的 `pptId`、`templateId` 和 `pptxProperty`。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "pptId": "pptId_xxx",
    "templateId": "templateId_xxx",
    "pptxProperty": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `pptId` | `string` | `pptId_xxx` | 字段用途请结合接口场景传入。 |
| `templateId` | `string` | `templateId_xxx` | 模板 ID。 |
| `pptxProperty` | `boolean` | `VALUE` | PPT 结构数据，用于编辑器回显或再次保存。 |
