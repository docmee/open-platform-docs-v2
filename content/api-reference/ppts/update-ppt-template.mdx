# 更换 PPT 模板

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/updatePptTemplate`
- 鉴权：请求头传 `token`

## 接口作用

为已经生成的 PPT 切换模板，并返回更新后的结构数据。

## 请求参数

```json
{
  "pptId": "ppt_xxx",
  "templateId": "template_xxx",
  "sync": false
}
```

- `sync=false`：默认异步更新，速度更快

## 响应说明

响应中会返回新的 `pptId`、`templateId` 和 `pptxProperty`。
