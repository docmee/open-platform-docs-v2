# 保存 PPT

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/savePptx`
- 鉴权：请求头传 `token`

## 接口作用

保存编辑后的 PPT 结构，并按需重新渲染 PPT 文件和封面。

## 请求参数

```json
{
  "id": "ppt_xxx",
  "drawPptx": true,
  "drawCover": true,
  "pptxProperty": {}
}
```

## 说明

- 只想重渲染 PPT 文件时，可只传 `drawPptx=true`
- 如果没有最新的 `pptxProperty`，可不传该字段
- 成功后会返回最新的 `pptInfo`
