# 下载自定义模板

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/downloadTemplate">https://open.docmee.cn/api/ppt/downloadTemplate</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

获取自定义模板详情和下载地址。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/downloadTemplate \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "id": "template_xxx"
}'
```

## 请求参数

```json
{
  "id": "template_xxx"
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `string` | 否 | `template_xxx` | 业务对象 ID。 |

## 返回内容

响应中通常包含 `id`、`subject`、`coverUrl`、`fileUrl`、`createTime`。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "id_xxx",
    "subject": "VALUE",
    "coverUrl": "VALUE",
    "fileUrl": "VALUE",
    "createTime": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `id` | `string` | `id_xxx` | 对象唯一标识。 |
| `subject` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
| `coverUrl` | `string` | `VALUE` | 封面图地址。 |
| `fileUrl` | `string` | `VALUE` | 文件下载地址。 |
| `createTime` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
