# 上传自定义模板

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/uploadTemplate`
- 鉴权：请求头传 `token`
- 请求体：`multipart/form-data`

## 接口作用

上传一个用户自定义模板，或通过 `templateId` 覆盖已有模板。

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `type` | `int` | 是 | 用户模板固定传 `4` |
| `file` | `File` | 是 | 仅支持 `pptx`，建议标准尺寸 960x540 |
| `templateId` | `string` | 否 | 传入时表示覆盖已有模板 |

## 说明

- 上传后系统会自动做模板学习与标注
- 覆盖公共模板时需要改用 `Api-Key` 调用
- 封面访问需要拼接 `token`
