# 设置为公共模板

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/updateUserTemplate">https://open.docmee.cn/api/ppt/updateUserTemplate</InlineCode>
- 鉴权：请求头传 `Api-Key`

## 接口作用

将自定义模板设置为当前 `Api-Key` 账号级别的公共模板，使该 `Api-Key` 下创建的所有 Token 都可见。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/updateUserTemplate \
  --header "Content-Type: application/json" \
  --header "Api-Key: YOUR_API_KEY" \
  --data '{
  "templateId": "template_xxx",
  "isPublic": true
}'
```

## 请求参数

```json
{
  "templateId": "template_xxx",
  "isPublic": true
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `templateId` | `string` | 否 | `template_xxx` | 模板 ID。 |
| `isPublic` | `boolean` | 否 | `true` | 字段用途请结合接口场景传入。 |

## 说明

这里的“公共模板”是账号级共享，不是平台全局公共模板。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `data` | `object` | `{...}` | 业务数据对象。 |
