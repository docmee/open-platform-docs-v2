# 修改模板属性

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/updateTemplate">https://open.docmee.cn/api/ppt/updateTemplate</InlineCode>
- 鉴权：请求头传 `Api-Key`

## 接口作用

修改自定义模板名称。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/updateTemplate \
  --header "Content-Type: application/json" \
  --header "Api-Key: YOUR_API_KEY" \
  --data '{
  "id": "template_xxx",
  "name": "新的模板名称"
}'
```

## 请求参数

```json
{
  "id": "template_xxx",
  "name": "新的模板名称"
}
```



## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `id` | `string` | 否 | `template_xxx` | 业务对象 ID。 |
| `name` | `string` | 否 | `新的模板名称` | 字段用途请结合接口场景传入。 |

## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "id": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `data` | `object` | `{...}` | 业务数据对象。 |
