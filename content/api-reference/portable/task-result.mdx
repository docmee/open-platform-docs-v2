# 查询便携版结果

## 接口信息

- 方法：`POST`
- 路径：`https://open.docmee.cn/v2/api/ppt/portable/taskResult`
- 鉴权：请求头传 `token`
- 请求体：`application/json`

## 接口作用

查询便携版任务的当前执行状态与最终结果。适用于以下场景：

- 创建任务时使用了 `stream=false`
- SSE 连接中断后恢复结果查询
- 业务侧需要主动轮询任务进度

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `taskId` | `string` | 是 | 创建便携版任务后返回的任务 ID |

## 返回字段

| 字段 | 说明 |
| --- | --- |
| `taskId` | 任务 ID |
| `status` | 任务状态：`pending`、`processing`、`success`、`failed` |
| `progress` | 当前进度，可选 |
| `step` | 当前执行阶段，可选 |
| `pptId` | 成功时返回 |
| `fileUrl` | 成功时返回，PPT 下载地址 |
| `coverUrl` | 成功时返回，封面图地址 |
| `previewUrl` | 成功时返回，预览地址 |
| `errorCode` | 失败时返回 |
| `errorMessage` | 失败时返回 |

```java
import java.net.http.HttpRequest;

public class PortableTaskResultExample {
  public static void main(String[] args) throws Exception {
    String body = """
      {

      }
      """;

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://open.docmee.cn/v2/api/ppt/portable/taskResult"))

      .header("token", "YOUR_TEMP_TOKEN")
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}
```

## 响应示例

处理中：

```json
{
  "code": 0,
  "data": {
    "taskId": "PORTABLE_TASK_ID",
    "status": "processing",
    "progress": 65,
    "step": "generate_ppt"
  }
}
```

成功：

```json
{
  "code": 0,
  "data": {
    "taskId": "PORTABLE_TASK_ID",
    "status": "success",
    "progress": 100,
    "step": "completed",
    "pptId": "PPT_ID",
    "fileUrl": "https://example.com/demo.pptx",
    "coverUrl": "https://example.com/cover.png",
    "previewUrl": "https://example.com/preview"
  }
}
```

