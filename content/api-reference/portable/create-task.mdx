# 创建便携版任务

## 接口信息

- 方法：`POST`
- 路径：`https://open.docmee.cn/v2/api/ppt/portable/createTask`
- 鉴权：请求头传 `token`
- 请求体：支持 `application/json` 和 `multipart/form-data`

## 接口作用

创建一个便携版 PPT 生成任务。服务端会在内部串联标准版的任务创建、内容生成、PPT 生成和下载地址生成流程。

如果传入 `stream=true`，接口会直接以 `SSE` 形式返回生成过程；否则先返回 `taskId`，后续通过 [`taskResult`](/api-reference/portable/task-result) 查询结果。

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `type` | `1-7` | 是 | 任务类型。`1` 智能生成；`2` 上传文件；`3` 思维导图；`4` Word 精准转 PPT；`5` 网页链接；`6` 粘贴文本；`7` Markdown 大纲 |
| `content` | `string` | 否 | 根据 `type` 传主题、网页链接、文本内容或 Markdown |
| `file` | `File[]` | 否 | 上传文件时使用，仅 `multipart/form-data` 支持，文件数不超过 5 个，总大小不超过 50M |
| `templateId` | `string` | 是 | 最终生成 PPT 使用的模板 ID |
| `stream` | `boolean` | 否 | 是否使用 SSE 长连接返回生成过程，默认 `false` |
| `length` | `short \| medium \| long` | 否 | 内容长度偏好 |
| `scene` | `string` | 否 | 演示场景 |
| `audience` | `string` | 否 | 受众类型 |
| `lang` | `string` | 否 | 输出语言 |
| `prompt` | `string` | 否 | 附加要求，建议小于 50 字 |

## 请求体说明

| Content-Type | 说明 |
| --- | --- |
| `application/json` | 不需要上传文件时使用 |
| `multipart/form-data` | 需要上传文件时使用 |

## 特别说明

- `prompt` 仅在任务类型为 `1`、`2`、`5`、`6` 时生效
- `type=2`、`type=4` 通常通过 `file` 上传原始文件
- `type=5` 时，`content` 传网页链接
- `type=7` 时，`content` 传 Markdown 大纲

package main

import (
  "bytes"
  "fmt"
  "io"
  "mime/multipart"
  "net/http"
  "os"

func main() {
  var body bytes.Buffer
  writer := multipart.NewWriter(&body)
  writer.WriteField("type", "2")
  writer.WriteField("templateId", "TEMPLATE_ID")
  writer.WriteField("stream", "false")
  writer.WriteField("length", "medium")
  writer.WriteField("scene", "产品介绍")
  writer.WriteField("audience", "客户")
  writer.WriteField("lang", "zh")
  writer.WriteField("prompt", "请突出重点结论")

  file, _ := os.Open("demo.docx")
  defer file.Close()

  part, _ := writer.CreateFormFile("file", "demo.docx")
  io.Copy(part, file)
  writer.Close()

  req, _ := http.NewRequest("POST", "https://open.docmee.cn/v2/api/ppt/portable/createTask", &body)
  req.Header.Set("token", "YOUR_TEMP_TOKEN")
  req.Header.Set("Content-Type", writer.FormDataContentType())

  resp, _ := http.DefaultClient.Do(req)
  defer resp.Body.Close()

  data, _ := io.ReadAll(resp.Body)
  fmt.Println(string(data))
}

## 响应方式

### 非流式响应

当 `stream=false` 或不传时，返回示例：

```json
{
  "code": 0,
  "data": {
    "taskId": "PORTABLE_TASK_ID",
    "status": "pending",
    "message": "task created"
  }
}
```

### SSE 响应

当 `stream=true` 时，接口返回 `text/event-stream`。示例：

```text
event: message
data: {"taskId":"PORTABLE_TASK_ID","status":"processing","step":"create_task","progress":10}

event: message
data: {"taskId":"PORTABLE_TASK_ID","status":"processing","step":"generate_content","progress":55}

event: message
data: {"taskId":"PORTABLE_TASK_ID","status":"processing","step":"generate_ppt","progress":85}

event: message
data: {"taskId":"PORTABLE_TASK_ID","status":"success","progress":100,"pptId":"PPT_ID","fileUrl":"https://example.com/demo.pptx"}
```
