# 异步生成内容并产出 PPT

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/generateContent`
- 鉴权：请求头传 `token`

## 接口作用

在生成内容的同时异步产出 PPT，并在流式响应中返回实时的 `pptId` 与进度信息。

## 请求参数

```json
{
  "templateId": "template_xxx",
  "outlineMarkdown": "# 主题\n## 章节\n### 页面标题",
  "asyncGenPptx": true,
  "prompt": "加强专业感",
  "dataUrl": null
}
```

## 关键说明

- `asyncGenPptx` 必须为 `true`
- 流式响应中会持续出现 `pptId`、`total`、`current`
- 每次收到新的 `pptId` 时，可调用 [`asyncPptInfo`](/api-reference/v1/async-ppt-info) 拉取最新渲染数据
