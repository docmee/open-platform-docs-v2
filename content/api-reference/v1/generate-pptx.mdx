# 生成 PPT（V1）

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/generatePptx">https://open.docmee.cn/api/ppt/generatePptx</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据完整 Markdown 内容生成 PPT。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/generatePptx \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "templateId": "template_xxx",
  "pptxProperty": false,
  "outlineContentMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题\n- 内容",
  "notes": ["备注一", "备注二"]
}'
```

## 请求参数

```json
{
  "templateId": "template_xxx",
  "pptxProperty": false,
  "outlineContentMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题\n- 内容",
  "notes": ["备注一", "备注二"]
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `templateId` | `string` | 否 | `template_xxx` | 模板 ID。 |
| `pptxProperty` | `boolean` | 否 | `false` | 字段用途请结合接口场景传入。 |
| `outlineContentMarkdown` | `string` | 否 | `# 主题 / ## 章节 / ### 页面标题 / #### 内容标题 / - 内容` | Markdown 格式的内容结构。 |
| `notes` | `array` | 否 | `["备注一","备注二"]` | 字段用途请结合接口场景传入。 |

## 响应结果

返回 `pptInfo`，其中通常包含：

- `id`
- `subject`
- `coverUrl`
- `fileUrl`
- `templateId`
- `pptxProperty`

## 适用场景

适合已经在 `V1` 链路中拿到完整 Markdown 内容，且希望继续沿用既有生成逻辑的系统。


## 响应示例

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "pptInfo": "VALUE",
    "id": "id_xxx",
    "subject": "VALUE",
    "coverUrl": "VALUE",
    "fileUrl": "VALUE",
    "templateId": "templateId_xxx",
    "pptxProperty": "VALUE",
    "V1": "VALUE"
  }
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 通用响应状态码，0 表示成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `pptInfo` | `object` | `VALUE` | 字段用途请结合接口场景传入。 |
| `id` | `string` | `id_xxx` | 对象唯一标识。 |
| `subject` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
| `coverUrl` | `string` | `VALUE` | 封面图地址。 |
| `fileUrl` | `string` | `VALUE` | 文件下载地址。 |
| `templateId` | `string` | `templateId_xxx` | 模板 ID。 |
| `pptxProperty` | `boolean` | `VALUE` | PPT 结构数据，用于编辑器回显或再次保存。 |
| `V1` | `string` | `VALUE` | 字段用途请结合接口场景传入。 |
