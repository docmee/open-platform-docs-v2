# 生成 PPT（V1）

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/generatePptx`
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据完整 Markdown 内容生成 PPT。

## 请求参数

```json
{
  "templateId": "template_xxx",
  "pptxProperty": false,
  "outlineContentMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题\n- 内容",
  "notes": ["备注一", "备注二"]
}
```

## 响应结果

返回 `pptInfo`，其中通常包含：

- `id`
- `subject`
- `coverUrl`
- `fileUrl`
- `templateId`
- `pptxProperty`

## 适用场景

适合已经在 `V1` 链路中拿到完整 Markdown 内容，且希望继续沿用既有生成逻辑的系统。
