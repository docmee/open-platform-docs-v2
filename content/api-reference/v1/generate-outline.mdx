# 生成大纲（V1）

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/generateOutline">https://open.docmee.cn/api/ppt/generateOutline</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据主题或 `dataUrl` 流式生成 PPT 大纲。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/generateOutline \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "stream": true,
  "length": "medium",
  "lang": "zh",
  "prompt": "突出商业价值",
  "subject": "AI 未来发展",
  "dataUrl": "https://xxx"
}'
```

## 请求参数

```json
{
  "stream": true,
  "length": "medium",
  "lang": "zh",
  "prompt": "突出商业价值",
  "subject": "AI 未来发展",
  "dataUrl": "https://xxx"
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `stream` | `boolean` | 否 | `true` | 是否启用流式返回。 |
| `length` | `string` | 否 | `medium` | 字段用途请结合接口场景传入。 |
| `lang` | `string` | 否 | `zh` | 输出语言。 |
| `prompt` | `string` | 否 | `突出商业价值` | 附加生成要求。 |
| `subject` | `string` | 否 | `AI 未来发展` | 字段用途请结合接口场景传入。 |
| `dataUrl` | `string` | 否 | `https://xxx` | 字段用途请结合接口场景传入。 |

## 响应方式

返回 `event-stream`，状态含义：

- `1`：解析文件
- `3`：生成中
- `4`：完成

## 适用场景

如果你的历史系统基于 `V1` 的“大纲 -> 内容 -> PPT”链路构建，可继续使用该接口。


## 响应示例

```text
event: message
data: {"status":3,"text":"生成中..."}

event: message
data: {"status":4,"markdown":"# 主题"}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `event` | `string` | `message` | SSE 事件名。 |
| `data.status` | `number` | `3` | 当前生成状态。常见值如 1、3、4。 |
| `data.text` | `string` | `生成中...` | 当前阶段返回的文本片段或提示。 |
| `data.markdown` | `string` | `# 主题` | 生成完成后返回的 Markdown 内容。 |
