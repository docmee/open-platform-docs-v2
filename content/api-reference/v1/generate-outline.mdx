# 生成大纲（V1）

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/generateOutline`
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据主题或 `dataUrl` 流式生成 PPT 大纲。

## 请求参数

```json
{
  "stream": true,
  "length": "medium",
  "lang": "zh",
  "prompt": "突出商业价值",
  "subject": "AI 未来发展",
  "dataUrl": "https://xxx"
}
```

## 响应方式

返回 `event-stream`，状态含义：

- `1`：解析文件
- `3`：生成中
- `4`：完成

## 适用场景

如果你的历史系统基于 `V1` 的“大纲 -> 内容 -> PPT”链路构建，可继续使用该接口。
