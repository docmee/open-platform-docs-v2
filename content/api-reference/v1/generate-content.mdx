# 生成大纲内容（V1）

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/generateContent">https://open.docmee.cn/api/ppt/generateContent</InlineCode>
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据大纲 Markdown 生成完整内容，适合在大纲确认后继续生成正文。

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/generateContent \
  --header "Content-Type: application/json" \
  --header "token: YOUR_TEMP_TOKEN" \
  --data '{
  "stream": true,
  "outlineMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题",
  "asyncGenPptx": false,
  "lang": "zh",
  "prompt": "表达更简洁",
  "dataUrl": null
}'
```

## 请求参数

```json
{
  "stream": true,
  "outlineMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题",
  "asyncGenPptx": false,
  "lang": "zh",
  "prompt": "表达更简洁",
  "dataUrl": null
}
```


## 请求字段说明

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `stream` | `boolean` | 否 | `true` | 是否启用流式返回。 |
| `outlineMarkdown` | `string` | 否 | `# 主题 / ## 章节 / ### 页面标题 / #### 内容标题` | Markdown 格式的内容结构。 |
| `asyncGenPptx` | `boolean` | 否 | `false` | 字段用途请结合接口场景传入。 |
| `lang` | `string` | 否 | `zh` | 输出语言。 |
| `prompt` | `string` | 否 | `表达更简洁` | 附加生成要求。 |
| `dataUrl` | `string` | 否 | `null` | 字段用途请结合接口场景传入。 |

## 关键参数

- `outlineMarkdown`：大纲 Markdown
- `asyncGenPptx=false`：表示同步内容生成模式
- `dataUrl`：可选，来自 [`parseFileData`](/api-reference/v1/parse-file-data)

## 响应方式

返回 `event-stream`，结束后可得到完整内容并继续调用 [`generatePptx`](/api-reference/v1/generate-pptx)。


## 响应示例

```text
event: message
data: {"status":3,"text":"生成中..."}

event: message
data: {"status":4,"markdown":"# 主题"}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `event` | `string` | `message` | SSE 事件名。 |
| `data.status` | `number` | `3` | 当前生成状态。常见值如 1、3、4。 |
| `data.text` | `string` | `生成中...` | 当前阶段返回的文本片段或提示。 |
| `data.markdown` | `string` | `# 主题` | 生成完成后返回的 Markdown 内容。 |
