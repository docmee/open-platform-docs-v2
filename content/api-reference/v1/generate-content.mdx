# 生成大纲内容（V1）

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/generateContent`
- 鉴权：请求头传 `token`

## 接口作用

`V1` 兼容接口。根据大纲 Markdown 生成完整内容，适合在大纲确认后继续生成正文。

## 请求参数

```json
{
  "stream": true,
  "outlineMarkdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题",
  "asyncGenPptx": false,
  "lang": "zh",
  "prompt": "表达更简洁",
  "dataUrl": null
}
```

## 关键参数

- `outlineMarkdown`：大纲 Markdown
- `asyncGenPptx=false`：表示同步内容生成模式
- `dataUrl`：可选，来自 [`parseFileData`](/api-reference/v1/parse-file-data)

## 响应方式

返回 `event-stream`，结束后可得到完整内容并继续调用 [`generatePptx`](/api-reference/v1/generate-pptx)。
