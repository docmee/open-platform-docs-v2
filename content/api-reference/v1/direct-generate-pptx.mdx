# 直接生成 PPT

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/directGeneratePptx`
- 鉴权：请求头传 `token`

## 接口作用

一步式直接生成 PPT，适合快速体验或不想拆成大纲与内容两个阶段的场景。

## 请求参数

```json
{
  "stream": false,
  "templateId": "template_xxx",
  "pptxProperty": false,
  "length": "medium",
  "lang": "zh",
  "prompt": "风格更专业",
  "subject": "AI 未来发展",
  "dataUrl": "https://xxx"
}
```

## 说明

- `subject` 与 `dataUrl` 可任选其一，也可同时提供
- 非流式返回时会直接给出 `pptInfo`
- 初期验证能力时可以使用，但新正式接入仍推荐 `V2`
