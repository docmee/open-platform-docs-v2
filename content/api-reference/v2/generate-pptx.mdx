# 生成 PPT

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/v2/generatePptx`
- 鉴权：请求头传 `token`

## 接口作用

根据任务 ID、模板 ID 和最终 Markdown 内容生成 PPT 作品。

## 请求参数

```json
{
  "id": "task_xxx",
  "templateId": "template_xxx",
  "markdown": "# 主题\n## 章节\n### 页面标题\n#### 内容标题\n- 内容"
}
```

- `id`：任务 ID
- `templateId`：模板 ID，通常来自模板接口
- `markdown`：最终用于生成 PPT 的完整内容

## 响应示例

```json
{
  "code": 0,
  "data": {
    "pptInfo": {
      "id": "ppt_xxx",
      "subject": "AI 办公趋势",
      "coverUrl": "https://xxx.png",
      "templateId": "template_xxx",
      "pptxProperty": "base64-gzip-json"
    }
  }
}
```

## 后续操作

- 下载文件：调用 [`downloadPptx`](/api-reference/ppts/download-pptx)
- 加载数据：调用 [`loadPptx`](/api-reference/ppts/load-pptx)
- 二次编辑后保存：调用 [`savePptx`](/api-reference/ppts/save-pptx)
