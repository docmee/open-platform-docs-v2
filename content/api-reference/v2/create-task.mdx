# 创建任务

## 接口信息

- 方法：<HttpMethodBadge method="POST" />
- 路径：<InlineCode value="https://open.docmee.cn/api/ppt/v2/createTask">https://open.docmee.cn/api/ppt/v2/createTask</InlineCode>
- 鉴权：请求头传 <InlineCode>token</InlineCode>
- 请求体：<InlineCode>multipart/form-data</InlineCode>

## 接口作用

创建一个新的 V2 生成任务。后续调用 <InlineCode>generateContent</InlineCode>、<InlineCode>updateContent</InlineCode> 和 <InlineCode>generatePptx</InlineCode> 时，都需要先拿到这里返回的任务 ID。

<ImportantNote title="重要信息">
  <p>
    <strong>必须先创建任务，再进行内容生成或 PPT 生成。</strong> 当 <InlineCode>type</InlineCode> 对应上传文件场景时，通常需要同时传入
    <InlineCode>file</InlineCode>。单次最多上传 <strong>5 个文件</strong>，总大小建议控制在 <strong>50M</strong> 以内。
  </p>
</ImportantNote>

## 请求示例

```bash
curl --request POST \
  --url https://open.docmee.cn/api/ppt/v2/createTask \
  --header "token: YOUR_TEMP_TOKEN" \
  --form "type=1" \
  --form "content=请生成一份关于 AI 办公趋势的产品介绍 PPT"
```

## 请求参数

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| `type` | `number` | 是 | `1` | 任务类型。`1` 智能生成，`2` 上传文件，`3` 思维导图，`4` Word 精准转 PPT，`5` 网页链接，`6` 粘贴文本，`7` Markdown 大纲。 |
| `content` | `string` | 否 | `请生成一份关于 AI 办公趋势的产品介绍 PPT` | 根据 `type` 传主题、网页链接、粘贴文本或 Markdown 大纲。 |
| `file` | `File[]` | 否 | `demo.docx` | 上传文件场景使用。通常在 `type=2` 或 `type=4` 时传入。 |

## 参数说明

- <InlineCode>type=1</InlineCode> 时，<InlineCode>content</InlineCode> 一般传主题或生成要求。
- <InlineCode>type=2</InlineCode>、<InlineCode>type=4</InlineCode> 时，通常通过 <InlineCode>file</InlineCode> 上传原始文件。
- <InlineCode>type=5</InlineCode> 时，<InlineCode>content</InlineCode> 传网页 URL。
- <InlineCode>type=6</InlineCode> 时，<InlineCode>content</InlineCode> 传粘贴文本。
- <InlineCode>type=7</InlineCode> 时，<InlineCode>content</InlineCode> 传 Markdown 大纲。

## 响应示例

```json
{
  "data": {
    "id": "task_xxx"
  },
  "code": 0,
  "message": "操作成功"
}
```

## 响应字段说明

| 字段名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `code` | `number` | `0` | 响应状态码，`0` 表示请求成功。 |
| `message` | `string` | `操作成功` | 服务端返回的结果说明。 |
| `data` | `object` | `{"id":"task_xxx"}` | 业务数据对象。 |
| `data.id` | `string` | `task_xxx` | 新创建的任务 ID，后续生成内容与生成 PPT 都依赖该值。 |

package main

import (
  "bytes"

  "io"

func main() {
  var body bytes.Buffer
  writer := multipart.NewWriter(&body)
  writer.WriteField("type", "1")
  writer.WriteField("content", "请生成一份关于 AI 办公趋势的产品介绍 PPT")
  writer.Close()

  req, _ := http.NewRequest("POST", "https://open.docmee.cn/api/ppt/v2/createTask", &body)
  req.Header.Set("token", "YOUR_TEMP_TOKEN")
  req.Header.Set("Content-Type", writer.FormDataContentType())

  resp, _ := http.DefaultClient.Do(req)
  defer resp.Body.Close()
  data, _ := io.ReadAll(resp.Body)
  fmt.Println(string(data))
}

import java.io.IOException;
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class CreateTaskExample {
  public static void main(String[] args) throws IOException, InterruptedException {
    String boundary = "----DocmeeBoundary";
    String body = "--" + boundary + "\\r\\n"
      + "Content-Disposition: form-data; name=\\"type\\"\\r\\n\\r\\n1\\r\\n"
      + "--" + boundary + "\\r\\n"
      + "Content-Disposition: form-data; name=\\"content\\"\\r\\n\\r\\n请生成一份关于 AI 办公趋势的产品介绍 PPT\\r\\n"
      + "--" + boundary + "--\\r\\n";

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://open.docmee.cn/api/ppt/v2/createTask"))
      .header("token", "YOUR_TEMP_TOKEN")
      .header("Content-Type", "multipart/form-data; boundary=" + boundary)
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}
