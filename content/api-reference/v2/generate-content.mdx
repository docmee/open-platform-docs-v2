# 生成大纲内容

## 接口信息

- 方法：`POST`
- 路径：`/api/ppt/v2/generateContent`
- 鉴权：请求头传 `token`

## 接口作用

根据任务 ID 为当前任务生成完整的大纲与内容。推荐使用流式响应，以便实时展示生成过程。

## 请求参数

```json
{
  "id": "task_xxx",
  "stream": true,
  "length": "medium",
  "scene": "产品介绍",
  "audience": "客户",
  "lang": "zh",
  "prompt": "语气专业"
}
```

- `id`：创建任务后获得的任务 ID
- `stream`：是否流式返回，默认 `true`
- `length`：`short`、`medium`、`long`
- `scene`：演示场景
- `audience`：受众类型
- `lang`：输出语言
- `prompt`：附加要求，小于 50 字

## 特别说明

`prompt` 仅在任务类型为 `1`、`2`、`5`、`6` 时生效，其他任务类型会忽略该字段。

## 响应方式

- `event-stream`：逐步返回生成文本片段与状态
- 结束时返回完整结构树结果，供后续生成 PPT 或继续修改

## 典型状态

- `3`：生成中
- `4`：生成完成

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class GenerateContentExample {
  public static void main(String[] args) throws Exception {
    String body = """
      {
        "id": "TASK_ID",
        "stream": true,
        "length": "medium",
        "scene": "产品介绍",
        "audience": "客户",
        "lang": "zh",
        "prompt": "语气专业，适合演示"
      }
      """;

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://docmee.cn/api/ppt/v2/generateContent"))
      .header("Content-Type", "application/json")
      .header("token", "YOUR_TEMP_TOKEN")
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}
```
