# 获取生成选项

## 接口信息

- 方法：`GET`
- 路径：`/api/ppt/v2/options`
- 鉴权：请求头传 `token`

## 接口作用

返回 V2 生成内容时可选的语言、场景、受众等配置项，适合在前端表单中动态展示。

## 返回内容

响应中会包含至少以下字段：

- `lang`：支持的语言列表
- `scene`：演示场景列表
- `audience`：受众列表

## 使用说明

- 该接口支持国际化，可在 URL 携带 `lang` 参数
- 推荐在调用 `generateContent` 前先拉取一次，避免前端写死枚举值
