# 使用指南

本文用于帮助首次接触 Docmee Open Platform 的团队快速理解平台能力、接入方式和推荐阅读路径。

## 平台可以做什么

- 通过 API 生成、编辑、下载和管理 PPT
- 通过新版 UI SDK 直接把完整工作台嵌入业务系统
- 通过模板、记录、文件和开放能力接口补齐不同业务链路

## 推荐阅读顺序

1. 如果你要直接内嵌前端工作台，先看 [UI 接入](/ui-integration)
2. 如果你要从后端接口开始联调，先看 [快速开始](/getting-started)
3. 如果你已经知道自己要调哪些接口，直接进入 [API 参考](/api-reference)
4. 如果你在评估更细分的能力形态，再看 [开放能力](/open-capabilities)

## 如何选择接入方式

- 追求最短集成路径：优先用 [UI 接入](/ui-integration)
- 需要完全控制生成链路和中间结果：优先用 [API 参考](/api-reference)
- 需要特定场景能力，如 HTML 转 PPTX：进入 [开放能力](/open-capabilities)

## 下一步

- 开始嵌入 UI：阅读 [新版 UI 接入](/ui-integration)
- 开始跑 API：阅读 [快速开始](/getting-started)
- 直接查接口：阅读 [全部 API 接口](/api-reference/all-api)
