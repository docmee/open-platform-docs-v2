# 如何使用本文档中心


本文档中心存在的目的是为了帮助您更好地接入我们的产品「文多多AiPPT」。



## 关于文多多 AiPPT

文多多 AiPPT，基于人工智能大模型，输入主题、资料、Word 文档等，一键生成精美 PPT。它设计精巧，配色高级，排版合理，元素搭配和谐，质感十足。无论是职场汇报还是校园展示，都能让你轻松俘获观众的心。

文多多 AiPPT 开放平台，为开发者提供便捷的 API 与 UI（Iframe）接入方式。API 调用轻松集成，无需繁琐申请，文档公开透明。UI 接入通过 iframe 方案，简单几步即可将文多多 AiPPT 嵌入业务系统。平台采用先进 AI 技术，支持原生图表、动画、3D 特效等复杂 PPT 的解析和渲染，还提供海量精品模板，满足个性化需求。无论是专业技术人员还是普通开发者，都能快速上手，为项目增添智能 PPT 生成能力，提升办公效率与创意。
