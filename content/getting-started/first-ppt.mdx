# 生成第一份 PPT

这一节只走一条最短路径：使用 `V2` 接口，从主题创建任务并生成第一份 PPT。

## 第 1 步：创建调用 Token

先在服务端使用 `Api-Key` 调用 [`createApiToken`](/api-reference/auth/create-api-token)。

<ApiCodeTabs
  node={`const response = await fetch('https://docmee.cn/api/user/createApiToken', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Api-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    uid: 'demo-user',
    limit: 10,
    timeOfHours: 2
  })
})

const result = await response.json()
console.log(result)`}
  python={`import requests

response = requests.post(
    'https://docmee.cn/api/user/createApiToken',
    headers={
        'Content-Type': 'application/json',
        'Api-Key': 'YOUR_API_KEY'
    },
    json={
        'uid': 'demo-user',
        'limit': 10,
        'timeOfHours': 2
    }
)

print(response.json())`}
  go={`package main

import (

拿到返回的 `token` 后，后续请求都在 Header 中传入：

```http
token: YOUR_TEMP_TOKEN
```

## 第 2 步：创建生成任务

调用 [`createTask`](/api-reference/v2/create-task) 创建任务。这里以 `type=1` 的智能生成为例。

<ApiCodeTabs
  node={`const formData = new FormData()
formData.append('type', '1')
formData.append('content', '请生成一份关于 AI 办公趋势的产品介绍 PPT')

const response = await fetch('https://docmee.cn/api/ppt/v2/createTask', {
  method: 'POST',
  headers: {
    token: 'YOUR_TEMP_TOKEN'
  },
  body: formData
})

const result = await response.json()
console.log(result)`}
  python={`import requests

response = requests.post(
    'https://docmee.cn/api/ppt/v2/createTask',
    headers={'token': 'YOUR_TEMP_TOKEN'},
    data={
        'type': '1',
        'content': '请生成一份关于 AI 办公趋势的产品介绍 PPT'
    }
)

print(response.json())`}
  go={`package main

import (
  "bytes"
  "fmt"
  "io"
  "mime/multipart"
  "net/http"
)

func main() {
  var body bytes.Buffer
  writer := multipart.NewWriter(&body)
  writer.WriteField("type", "1")
  writer.WriteField("content", "请生成一份关于 AI 办公趋势的产品介绍 PPT")
  writer.Close()

  req, _ := http.NewRequest("POST", "https://docmee.cn/api/ppt/v2/createTask", &body)
  req.Header.Set("token", "YOUR_TEMP_TOKEN")
  req.Header.Set("Content-Type", writer.FormDataContentType())

  resp, _ := http.DefaultClient.Do(req)
  defer resp.Body.Close()

  data, _ := io.ReadAll(resp.Body)
  fmt.Println(string(data))
}`}
  java={`import java.io.IOException;
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class CreateTaskExample {
  public static void main(String[] args) throws IOException, InterruptedException {
    String boundary = "----DocmeeBoundary";
    String body = "--" + boundary + "\\r\\n"
      + "Content-Disposition: form-data; name=\\"type\\"\\r\\n\\r\\n1\\r\\n"
      + "--" + boundary + "\\r\\n"
      + "Content-Disposition: form-data; name=\\"content\\"\\r\\n\\r\\n请生成一份关于 AI 办公趋势的产品介绍 PPT\\r\\n"
      + "--" + boundary + "--\\r\\n";

    HttpRequest request = HttpRequest.newBuilder()
      .uri(URI.create("https://docmee.cn/api/ppt/v2/createTask"))
      .header("token", "YOUR_TEMP_TOKEN")
      .header("Content-Type", "multipart/form-data; boundary=" + boundary)
      .POST(HttpRequest.BodyPublishers.ofString(body))
      .build();

    HttpResponse<String> response = HttpClient.newHttpClient()
      .send(request, HttpResponse.BodyHandlers.ofString());

    System.out.println(response.body());
  }
}`}
/>

成功后会返回任务 `id`。

## 第 4 步：选择模板

如果你已经有模板 ID，可以直接使用；如果没有，可以先调用：

- [`templates`](/api-reference/templates/list-templates) 分页查模板
- [`randomTemplates`](/api-reference/templates/random-templates) 随机拿一批模板

## 常见建议

- 新接入优先使用 `V2`，不要从 `V1` 开始
- 生产环境由服务端创建 Token，不要在前端暴露 `Api-Key`
- UI 集成建议为 Token 设置 `limit`
- 资源访问需要在 URL 上追加 `?token=xxx`
